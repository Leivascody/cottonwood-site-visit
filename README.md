- `video/cottonwood-aerial.mp4` — looping drone walkthrough video

## Hosting on GitHub Pages

1. Create a new GitHub repository (public).
2. Push this folder to the repo:
   ```bash
   git init -b main
   git add .
   git commit -m "Cottonwood site visit report"
   git remote add origin https://github.com/<your-user>/<repo-name>.git
   git push -u origin main
   ```
3. In the repo settings, go to **Pages** and set the source to `main` branch / root.
4. Your report will be live at `https://<your-user>.github.io/<repo-name>/` within a minute.
